# 🌿 MsituGuard - Tree Survival Prediction & Environmental Reporting

**AI-Powered Tree Survival Prediction for Kenya**

Get accurate tree survival predictions for any Kenyan county and report environmental issues in your community.

## 🎯 What You Can Do

### 🤖 Tree Survival Prediction
- Get AI recommendations for tree species that will survive in your county
- See survival rates based on location, season, and planting method
- Receive care instructions powered by MISTRAL AI

### 📱 Environmental Reporting
- Report environmental issues with photos
- Track your reports and see community impact
- Organizations can verify and respond to reports

### 📊 Track Your Impact
- Register your tree planting activities
- View your environmental contributions
- Earn points and badges for verified actions

## 🚀 Technology

- **AI Model**: GradientBoosting Classifier + MISTRAL AI
- **Framework**: Django 5.0.3
- **Database**: All 47 Kenyan counties with species data
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap

## 🛠️ Installation

```bash
# Clone and setup
git clone https://github.com/MsituGuard-Sigma/MsituGuard.git
cd MsituGuard
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
